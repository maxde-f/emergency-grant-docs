# Emergency Grant Management System - Architecture Documentation

Interactive architecture diagrams for the Emergency Grant Management System.

## Features

- Interactive Mermaid diagrams
- Grant cycle flow visualization
- AI & Blockchain integration touchpoints
- Detailed system architecture

## Deployment

This site is deployed on Vercel.

## Files

- `index.html` - Interactive HTML with Mermaid diagrams
- `architecture.md` - Markdown version with Mermaid diagrams
- `vercel.json` - Vercel deployment configuration

## Local Development

Simply open `index.html` in a web browser.
